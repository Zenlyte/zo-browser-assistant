🚀 **Zo Browser Assistant v1.2.1 & v1.2.2 — Combined Release**

Two big updates just landed! Streaming stability + Speed & keyboard shortcuts.

---

**✨ v1.2.1 — Persistent Streaming**

• **Uninterrupted Chat Handoff**: Start a query in popup, switch to sidebar mid-stream — response continues seamlessly!\
• **No More "No Response"**: Fixed edge case where responses appeared lost despite successful API calls\
• **Background Worker**: Core streaming logic moved to service worker to prevent UI-triggered cancellations

---

**⚡ v1.2.2 — Speed & Power-User Features**

**Faster Zo Workspace Files**

- 5-minute file tree cache = instant loading on revisit
- Refresh button forces fresh fetch when needed
- Per-directory caching (each folder cached independently)

**Keyboard Shortcuts** (Customizable at `chrome://extensions/shortcuts`)

- `Ctrl+Shift+Z` — Open Zo Sidebar
- `Ctrl+Shift+X` — Quick Save current page to Zo
- `Ctrl+Shift+O` — Open Zo Popup
- `Ctrl+Shift+A` — Activate "Save to Zo" in popup

---

**🛠️ How to Update**

1. Download the latest package: <https://github.com/Zenlyte/zo-browser-assistant> 
2. Overwrite your existing extension folder with the new package
3. Open `chrome://extensions/`
4. Find **Zo Browser Assistant**
5. Click the **Reload** icon 🔄
