**✨ What's New**

• **Clean separation** — Public repo works for everyone, your personal setup is just config
• **No more branch drift** — Bug fixes flow to everyone automatically

**🛠️ For Personal Users**
1. Open Extension Options
2. Scroll to "Custom Save Script (Optional)"
3. Enter: `/home/workspace/Automations/Scripts/save_url.ts`
4. Done! Click "Save to Zo" → your script runs automatically
