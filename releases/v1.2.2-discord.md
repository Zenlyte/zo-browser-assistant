🚀 **Zo Browser Assistant v1.2.2 Released**

Speed & Power-User Features!

**⚡ Faster Zo Workspace Files**
- File tree now caches for 5 minutes (instant loading on revisit)
- Refresh button forces fresh fetch when needed
- Per-directory caching (each folder cached independently)

**⌨️ Keyboard Shortcuts**
- `Ctrl+Shift+Z` — Open Zo Sidebar
- `Ctrl+Shift+S` — Quick Save current page to Zo
- `Ctrl+Shift+O` — Open Zo Popup
- `Ctrl+Shift+A` — Activate "Save to Zo" in popup

**How to Use Shortcuts:**
1. Go to `chrome://extensions/shortcuts`
2. Assign your preferred keys (or keep defaults above)
3. Use anywhere in Chrome!
