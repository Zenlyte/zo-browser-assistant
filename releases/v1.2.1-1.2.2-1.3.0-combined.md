🚀 **Zo Browser Assistant v1.3.0 — Major Release: Streaming, Speed & Personal Setup**

Three big updates landing together! This is the biggest release yet—streaming stability, speed improvements, keyboard shortcuts, and a complete architecture overhaul for personal customization.

---

**⚡ v1.2.1 — Persistent Streaming (FIXED)**

• **Uninterrupted Chat Handoff**: Start a query in popup, switch to sidebar mid-stream—response continues seamlessly! No more broken streams when switching contexts.
• **No More "No Response"**: Fixed the edge case where responses appeared lost despite successful API calls. Every response now reliably reaches your screen.
• **Background Worker Architecture**: Core streaming logic moved to service worker to prevent UI-triggered cancellations.

---

**🚀 v1.2.2 — Speed & Power-User Features**

**Instant Saves — Non-Blocking UI**
Hitting "Save" no longer blocks the interface. The extension immediately confirms receipt (under 1s) and hands heavy lifting (page summarization + storage) to a background Zo process. Close the tab instantly—Zo finishes server-side.

**Faster Zo Workspace Files**
- 5-minute file tree cache = instant loading on revisit
- Refresh button forces fresh fetch when needed  
- Per-directory caching (each folder cached independently)

**Keyboard Shortcuts** (Customizable at `chrome://extensions/shortcuts`)
- `Ctrl+Shift+Z` — Open Zo Sidebar
- `Ctrl+Shift+X` — Quick Save current page to Zo (with visual badge feedback: ⏳ processing → ✓ saved → ❌ error)
- `Ctrl+Shift+O` — Open Zo Popup
- `Ctrl+Shift+A` — Activate "Save to Zo" in popup

---

**🎉 v1.3.0 — Personal Setup Made Easy (NEW)**

**No More Hardcoded Paths!**
Big architecture update: the public repo now works for everyone out-of-the-box, while your personal setup lives purely in configuration.

**Custom Save Script in Settings**
• New "Custom Save Script (Optional)" field in Extension Options
• Enter your personal script path once: `/home/workspace/Automations/Scripts/save_url.ts`
• Extension uses it automatically—no code changes needed!

**What This Means:**
- ✅ Single codebase for everyone (public users get default behavior)
- ✅ Your personal setup is just config (script path in Settings)
- ✅ No more branch drift—bug fixes flow to everyone automatically
- ✅ Easy to share extension with others
- ✅ Clean separation: public repo stays generic, your customizations stay personal

**Behind the Scenes:**
The extension sends a generic "save this page" prompt to Zo. Want custom behavior? Add a Zo Rule that intercepts the prompt and runs your script—you can sync to Raindrop, push to Notion, auto-tag with your second brain, whatever your Zo setup supports.

**Power-User Tip:** Hook up TokenCut in your custom script to reduce token usage on each page processing!

---

**🛠️ How to Update**

1. Download latest package: https://github.com/Zenlyte/zo-browser-assistant
2. Overwrite existing extension folder with new package
3. Open `chrome://extensions/`
4. Find **Zo Browser Assistant**
5. Click **Reload** icon 🔄

**For Personal Users (v1.3.0):**
1. After updating, open Extension Options
2. Find "Custom Save Script (Optional)"
3. Enter: `/home/workspace/Automations/Scripts/save_url.ts`
4. Done! Your custom save flow is restored.
