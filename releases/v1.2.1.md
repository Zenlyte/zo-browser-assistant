# 🚀 Zo Browser Assistant v1.2.1 — Release Notes

v1.2.1 just dropped with a major stability fix for chat streaming!

### ✨ What's New
• **Persistent Streaming**: Handoff active chat streams between popup and sidebar seamlessly. You can now start a query in the popup and immediately switch to the sidebar without the response getting cancelled.
• **Reliability**: Fixed an edge case where "No response" would appear despite a successful API call.

### 🛠️ Fixes
• Moved core streaming logic to the background service worker to prevent UI-triggered cancellations.
• Improved synchronization between popup and sidebar when a stream is in progress.

**How to update:**
1. Open `chrome://extensions/`
2. Find **Zo Browser Assistant**
3. Click the **Reload** (refresh) icon 🔄

Enjoy! Let me know if you have feedback! 🙌
