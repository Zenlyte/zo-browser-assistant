Zo Browser Assistant

Configure your API key to get started

Get your API key from Settings, Advanced, Access Tokens

Loads all available models for this API key (native + BYOK). This default is used when popup/sidepanel opens.

Features

  • Save any webpage to your Zo workspace
  • Chat with Zo about the current page
  • Summarize articles and documents
  • Extract key information from pages
  • Ask questions about page content
